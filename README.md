# WCProbe — Advanced Domain & IP Analysis Tool

WCProbe is a modern web-based Domain and IP analysis tool developed by WhiteCoder. It helps users check domain IP addresses, reverse DNS details, SSL certificate status, MX records, geo-location data, ISP details, and more through a clean and responsive interface.

## Features

- Domain to IP lookup
- IP to domain / reverse DNS lookup
- SSL certificate checking
- SSL issuer and expiry date detection
- MX record lookup
- Geo-location details
- ISP and organization details
- Cloudflare detection
- Multiple domain/IP analysis
- TXT and CSV file upload support
- Drag and drop file input
- Recent search history
- Result filtering
- Copy resolved IP addresses
- Export results as CSV
- Print or save results as PDF
- Light and dark mode
- 3D Earth background using Three.js
- Fully responsive modern UI

## How to Use

1. Download the project ZIP file.
2. Extract the ZIP file using the password.
3. Place the project folder inside your local server directory.
4. Start Apache using XAMPP or another PHP server.
5. Open the project in your browser.
6. Enter a domain or IP address.
7. Click the Analyze button.
